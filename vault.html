What is Vault?

Vault is an implementation of "Time-Locked Wallets", originally by Radek Ostrofski - modified just to hold ETC.

WARNING: Do not deposit anything other than ETC to your wallet contracts!


Features
  • Create as many wallet contracts as you like
  • Deposit ETC to your wallets any time
  • Withdraw ETC only when it's time
  • Optionally assign a beneficiary other than yourself

FAQs

How does this work?

Arcadium Vault offers a Smart Contract Interface which allows you to create Time-locked Wallet Contracts - so, you can create a "Lockbox" that only opens for a particular address at a certain date and time! What's better, is you can choose the address which can open the wallet and receive its contents - deposit anytime, open only when it's time!

Can I withdraw from the Wallet before time is reached?

No. We would advise not even trying.

Can I add to the wallet balance before time is reached?

You can add ETC to the wallet balance at ANY time!
